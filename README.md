# *Simple Theater Seat Booking System*

---
#### To view, click on : [Live Link](https://saadferozee.github.io/Theater-ticketing-system)
---

### A simple single-page theater seat booking application built using:

* HTML
* CSS
* Vanilla JavaScript

---
## Required Features was:

- [x] Dynamically generates 100 seats.
- [x] Seat naming follows A1-A25, B1-B25, C1-C25, D1-D25.
- [x] Book a seat using the input field.
- [x] Supports case-insensitive input (a1 or A1).
- [x] Prevents duplicate bookings.
- [x] Shows alert when a seat is already booked.
- [x] Automatically clears the input after booking.
- [x] Uses localStorage to preserve bookings after page refresh.
- [x] Responsive layout using CSS Grid.
